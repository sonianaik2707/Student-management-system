# Student-management-system

A simple command-line Student Management System built using Python. This project stores student records in a CSV file and provides a menu-driven interface to manage them.

> 🚧 This project is currently under development.

## Features

- Menu-driven interface
- Add student records
- Delete student records using Roll Number
- Store records in a CSV file
- Basic input validation
- Object-Oriented Programming using a `Student` class

## Project Structure

```
student-management-system/
│── main.py          # Main menu and navigation
│── student.py       # Student-related functions
│── student.csv      # Stores student records
│── README.md
```
